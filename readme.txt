RAM cache, он же кэш первого уровня. Объекты данного кэша располагаются в оперативной памяти.
HDD cache, он же кэш второго уровня. Объекты данного кэша располагаются в памяти жёсткого диска.
В программе реализовано 3 алгоритма кэширования - lru, mru, lrr. 
Запуск программы возможен без каких-либо аргументов командной строки, в этом случае все переменные будут настроены по-умолчанию. Отчёт по процессу кэширования пишется в файл Cache.log, автоматически создаваемый в папке с программой.
Файлы для кэша 2-го уровня создаются в папке /Cached Files/cache_file_XXXXXXX.cache

Аргументы командной строки (могут быть написаны заглавными буквами):

	level1CacheEntries=целое_число или l1ce=целое_число
		целое_число > 0;	Максимальное число записей для кэша 1-го уровня (RAM кэша).

	level2CacheEntries=целое_число или l2ce=целое_число
		целое_число > 0;    	Максимальное число записей для кэша 2-го уровня (HDD кэша).

	cachekind=строка или ck=строка
		lru	- Least recently used	(Вытеснение давно использовавшегося)
		mru	- Most recently used	(Вытеснение недавно использовавшегося)
		lrr	- Least recently replaced (Вытеснение самого давнего)
			При указании строки, отличной от вышеуказанных, выставляется алгоритм mru.

	dr	- detailed report. Детализация информации о кэшировании. При наличии этого аргумента, в лог файл пишется вся информация о выполнении процесса работы с кэшем. При его отсутствии - только общая сводка (Summary). По умолчанию, детальная информация отсутвует.
